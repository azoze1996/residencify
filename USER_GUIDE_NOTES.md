# User Guide: Notes Feature

## How to Use Notes in Residencify

### Adding a Note to a Question

1. **During Practice:**
   - While viewing any question, look for the **Note button** in the top-right area
   - The button shows a sticky note icon 📝
   - On mobile: Tap the note icon
   - On desktop: Click the "Note" button

2. **In the Note Dialog:**
   - Type your note in the text area (up to 3,000 characters)
   - Add anything helpful:
     - Key concepts to remember
     - Mnemonics
     - Why you got it wrong
     - Related topics to review
     - Personal insights
   - Click **"Save Note"** when done

3. **Visual Feedback:**
   - The note button turns **blue** when a note exists
   - The icon fills in to show there's a note
   - You'll see a success message: "Note saved!"

### Editing an Existing Note

1. Click the **Note button** on any question with a note (blue button)
2. The dialog opens with your existing note loaded
3. Make your changes
4. Click **"Save Note"** to update
5. You'll see: "Note updated!"

### Viewing All Your Notes

1. Go to **Dashboard** → **Notes** (in the sidebar)
2. You'll see all your notes organized by:
   - Category
   - Question number
   - Date created
3. Use the **search bar** to find specific notes
4. Click any note to view the full text

### Deleting a Note

**From the Question:**
1. Open the note dialog
2. Clear all the text
3. Click "Save Note" (saves as empty, effectively deleting)

**From the Notes Page:**
1. Hover over a note card
2. Click the **trash icon** that appears
3. Confirm deletion
4. The note is permanently removed

### Privacy & Security

✅ **Your notes are completely private**
- Only you can see your notes
- Notes are not shared with other users
- Notes are not visible to instructors or admins
- Each note is tied to your user account

✅ **Notes are saved automatically**
- Stored in the database
- Persist across devices
- Available whenever you log in

### Tips for Effective Note-Taking

1. **Be Concise:** Focus on key points you need to remember
2. **Use Your Own Words:** This helps with retention
3. **Add Context:** Include why something is important
4. **Review Regularly:** Visit your Notes page to review
5. **Tag Questions:** Mention related topics for easy searching

### Keyboard Shortcuts

- **Esc:** Close note dialog
- **Ctrl/Cmd + Enter:** Save note (when in text area)

### Mobile Tips

- **Tap the note icon** in the question header
- Use your device keyboard for typing
- Notes sync across all your devices
- Swipe to scroll through long notes

### Troubleshooting

**Note button not visible?**
- Make sure you're in practice mode (viewing a question)
- Check that you're logged in
- Try refreshing the page

**Note not saving?**
- Check your internet connection
- Make sure the note isn't empty
- Try again - you'll see an error message if it fails

**Can't find a note?**
- Go to Dashboard → Notes
- Use the search feature
- Check if you're looking at the right category

**Character limit reached?**
- Maximum 3,000 characters per note
- You'll see a warning at 2,500 characters
- Consider breaking into multiple notes if needed
